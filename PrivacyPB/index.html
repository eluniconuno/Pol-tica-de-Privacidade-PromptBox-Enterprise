Política de Privacidade – PromptBox Enterprise

Última atualização: 2025

O PromptBox Enterprise é um aplicativo criado dentro da plataforma OpenAI GPTs. Ele não coleta, armazena ou compartilha dados pessoais fora da própria sessão.

Coleta de Dados

Não coletamos dados pessoais permanentes. As informações fornecidas pelo usuário são usadas apenas para processar respostas dentro da mesma sessão.

Uso de Dados

Os dados fornecidos são usados para criar prompts, templates, agentes e automações dentro da mesma sessão.

Armazenamento

Não armazenamos dados de forma persistente. Todo conteúdo segue as políticas de privacidade da OpenAI.

Compartilhamento

Nenhum dado é compartilhado com terceiros.

Segurança

A segurança é provida pela infraestrutura da OpenAI.

Contato

Para dúvidas, utilize o próprio ambiente OpenAI GPTs.
